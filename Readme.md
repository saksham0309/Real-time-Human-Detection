# Install the requirements using
```pip install -r Requiremtns.txt```

# To execute
1. For Picture

```python main.py - i <Path_to_image>```

2. For Video


```python main.py - v <Path_to_video>```

3. For Camera

```python main.py -c True```

4. To save the output:

```Python main.py -c True -o ‘file_name’```
